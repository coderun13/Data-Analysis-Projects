## 📖 Overview
This repository serves as a professional portfolio featuring **13 distinct data analysis projects**. Each project follows an industrial workflow: from raw data ingestion and cleaning to advanced statistical modeling and executive-level visualization.



## 🛠️ Technical Workflow
Every project in this repository implements the following "Industrial Standard" steps:
* **Data Cleaning:** Handling null values, deduplication, and parsing complex CSV errors (EOF errors).
* **Outlier Management:** Implementing the **Interquartile Range (IQR)** method to ensure statistical accuracy.
* **EDA:** Exploratory Data Analysis to uncover hidden patterns and correlations.
* **Visualization:** Creating professional-grade dashboards using `Seaborn` and `Matplotlib`.

## ⚙️ Installation & Usage

### 1. Clone the Repository
```bash
git clone https://github.com/coderun13/Data-Analysis-Projects.git
```

### 2. Set Up Environment
```bash
python -m venv venv
venv\Scripts\activate
```
### 3. Install Requirements
```bash
pip install pandas seaborn matplotlib numpy jupyter
```

### 4. Run the Analysis
```bash
# Launch Jupyter Notebook
jupyter notebook
Once Jupyter opens, navigate to any project folder and run the .ipynb file.
```
